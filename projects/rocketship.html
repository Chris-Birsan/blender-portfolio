Rocketship Launch

3D Model | Blender | 2024

About This Project

A stylized rocket ship launching from a cartoon planet with volumetric cloud effects. This project focused on whimsical design, vibrant colors, and creating a sense of motion and energy.

Techniques Used

  • Stylized low-poly modeling
  • Vibrant material design with gradient textures
  • Volumetric cloud simulation
  • Dynamic composition and framing

Tools & Software

  • Blender 3.6 - Modeling and rendering
  • Cycles - Volumetric rendering
